O objetivo com esse código é criar um local onde o usuário é cadastrado para receber serviços de entregas de produtos.
Dessa forma dados como endereço, CPF, número de telefone são necessários para garantir que o processo de entrega seja feito. Garantindo que casso algum problema ocorra seja possível entrar em contato com o mesmo usando múltiplas formas e garantir que a entrega seja feita no local correto.
